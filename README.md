# EBX Core Chat Base

Projet de base pour EBX Core Chat avec composants UI.

## Structure

- `ebx-core-app`: Module application core
- `ebx-core-ui`: Module UI avec composants TypeScript/React

## Développement

Ce projet sert de base pour le développement de fonctionnalités de chat dans EBX Core.

### Prérequis

- Java 17
- Node.js
- Maven

### Build

```bash
./mvnw clean install
```

### UI Development

```bash
cd ebx-core-ui
npm install
npm run dev
```
